lDaylightSavingTime could be:

  • 0: DaylightSavingTime NO present for the input time (ulSrcYear, ...)
  • 1: DaylightSavingTime present for the input time (ulSrcYear, ...)
  • -1: information about DaylightSavingTime not available The output pointers parameters (pulDestYear, pulDestMonth, ...) could be also the same addresses of the input parameters (ulSrcYear, ulSrcMonth, ...)
