Detailed Description

La classe ErrMsgBase rappresenta la classe di base per ciascun tipo di errore (e/o messaggio). Qualunque classe di errore (e/o messaggio) deve ereditare da ErrMsgBase. Essa realizza le funzioni di base comuni ad ogni classe di errore (e/o messaggio).
