The FileIO class is a collection of static methods just to hide the differences to access a File/Directory between different operative system.

Error FileIO::lockFile ( const char *  pLockPathName,
long  lSecondsToWaitIfAlreadyLocked,
int *  pfdLockFile 
)
static

Lock the file specified by the pLockPathName parameter and return its file descriptor. In case the file is already locked, this method wait for lSecondsToWaitIfAlreadyLocked seconds to see if the file is unlocked. This method returns the TOOLS_FILEIO_FILENOTRELEASED in case:

  1. lSecondsToWaitIfAlreadyLocked is major than 0 and the file is already locked after that the time is passed
  2. lSecondsToWaitIfAlreadyLocked is -1 and the file is already locked
